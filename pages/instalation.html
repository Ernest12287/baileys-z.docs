üü¢ Step 1: Installing Node.js

Node.js is the foundation of Baileys-z. We recommend the LTS (Long-Term Support) version because it's stable and maintained for security and bug fixes‚Äîperfect for production bots.

üì• Windows & macOS

  1. Visit the official Node.js download page
  2. Download the LTS version installer for your operating system
  3. Run the installer and follow the setup wizard
  4. Restart your terminal after installation

üêß Linux (Debian/Ubuntu/MX)

Open your terminal and run:

‚úÖ Verify Installation

üé• Step 2: Installing FFmpeg

FFmpeg is a powerful command-line tool that processes audio and video files. Baileys-z uses it to ensure media sent through WhatsApp is properly formatted and compatible across all devices.

ü§î Why FFmpeg? Without FFmpeg, your bot won't be able to send or process voice notes, videos, or audio files reliably. It's a must-have for media-rich bots.

üêß Linux (Debian/Ubuntu/MX)

Update package list 
sudo apt update
Install FFmpeg 
sudo apt install ffmpeg

üçé macOS

If you have Homebrew installed (a package manager for macOS):

Install FFmpeg via Homebrew 
brew install ffmpeg

Don't have Homebrew? Install it first: brew.sh

ü™ü Windows

Option 1: Using Chocolatey (Recommended)

Install FFmpeg via Chocolatey 
choco install ffmpeg

Don't have Chocolatey? Install it first: chocolatey.org/install

Option 2: Manual Installation

  1. Download FFmpeg from ffmpeg.org
  2. Extract the downloaded ZIP file to C:\ffmpeg
  3. Add FFmpeg to your system PATH:
    • Search for "Environment Variables" in Windows Settings
    • Click "Edit the system environment variables"
    • Click "Environment Variables" button
    • Under "System variables", find and select "Path"
    • Click "Edit" ‚Üí "New" ‚Üí Add C:\ffmpeg\bin
    • Click "OK" on all dialogs
    • Restart your terminal

‚úÖ Verify Installation

Confirm FFmpeg is installed by running:

ffmpeg -version

You should see version information displayed.

üìÅ Create Your Project
