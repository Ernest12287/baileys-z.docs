Installation OF Baileys-z

Before starting: Baileys-z is very modern; it requires some prerequisites to run.

  1. Node v18+ (Required for the runtime environment)
  2. FFmpeg (Required for handling and converting audio/video files)
  3. A Package Manager (npm, Yarn, or pnpm - used to install Baileys-z)
  4. qrcode-terminal (Recommended npm package for easy authentication)

Additional Ernest Tech House Packages

  1. ernest-logger (for logging bot details)
  2. And others will be added as we go on.

Now we shall begin

Installation of Node.js

To install Node.js, you visit their official website and make sure you download the **LTS version** as that will be the **Long-Term Support version**.

For those that use other operating systems like myself (MX Linux) and any other OS, you will install it using the terminal command: 

sudo apt install nodejs

Well then, after that, you open your terminal and either run `node -v` or `node --version`. An output should present itself of a Node version depending on the year and version you installed.

How to install FFmpeg

**FFmpeg** is a crucial command-line tool used by Baileys-z to properly format media (especially audio and video) so that it is sent reliably across all WhatsApp platforms. You must have this installed globally on your machine.

Installation on Linux (Debian/Ubuntu/MX)

Use your system's package manager, which is typically `apt`: 

sudo apt update
sudo apt install ffmpeg

Installation on macOS

If you have the **Homebrew** package manager installed (highly recommended for macOS development), use the following command: 

brew install ffmpeg

Installation on Windows

The easiest way is to use the **Chocolatey** package manager (if installed) or follow the official download guide to add FFmpeg to your system's PATH variable.

**Using Chocolatey:** 

choco install ffmpeg

**Verification:** To check if FFmpeg is correctly installed, open your terminal and run: 

ffmpeg -version

This should display the current version details.

Development Environment Setup

Recommended Code Editor (IDE)

While you can use any text editor you like, we strongly recommend using an Integrated Development Environment (IDE) to make development smoother, thanks to features like syntax highlighting, code completion, and integrated terminals.

The most popular and recommended IDE for Node.js development is **Visual Studio Code (VS Code)** by Microsoft.

To install VS Code:

  1. Visit the official VS Code download page.
  2. Download the installer appropriate for your operating system (Windows, macOS, or Linux).
  3. Run the installer and follow the on-screen instructions.

Future Announcement: Ernest Tech House is excited to announce that we are currently developing our own dedicated IDE optimized for the **Baileys-z** ecosystem! We aim to release this new development tool by the end of **2026**.

Installing Baileys-z and Dependencies

Now that you have the core system dependencies and your preferred IDE installed, we can set up your project and install **Baileys-z** and the supporting **Ernest Tech House** libraries.

Step 1: Initialize Your Project

Navigate to the directory where you want to create your bot project and initialize a new Node.js project. You can do this easily using the terminal built into VS Code: 

npm init -y

This creates a `package.json` file for tracking your dependencies.

Step 2: Install Baileys-z and Related Packages

We will use **npm** for this example. Replace `npm install` with `yarn add` or `pnpm add` if you prefer a different package manager. 

npm install baileys-z ernest-logger @peaseernest/system-info qrcode-terminal

This command installs:

  • **`baileys-z`**: The core WhatsApp API library.
  • **`ernest-logger`**: The official logging utility from Ernest Tech House.
  • **`@peaseernest/system-info`**: Another utility for tracking system details.
  • **`qrcode-terminal`**: The recommended package to display the QR code for authentication directly in your terminal.

You are now ready to start writing your first Baileys-z code!
