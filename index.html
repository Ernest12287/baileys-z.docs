What is Baileys-z?

Just like the official Baileys library (Baileys 7.0.0rc), it uses **WebSockets** to connect with WhatsApp. It is **lightweight** and very **easy to implement**.

This library is merely a fork of Baileys and includes some **speed improvements** and **minor tweaks**. Even though we depend so much on the official Baileys library for in-library updates, **our fork is aimed to be super fast and easy to use**, and also provides **easy communication** with us if we see an error or you as a user see an error.

Our library uses the latest version of Baileys: **V7.0.0rc**. Right now, that is the most community maintained version. The previous owner, Adjwashing, is no longer in charge of the library; it is now community owned.
